# Material Tailwind 使用笔记

[Material Tailwind](https://www.material-tailwind.com/) is a free and open-source components library for ReactJS and Tailwind CSS inspired by Material Design.

[Material Tailwind](https://www.material-tailwind.com/) 是一个免费的开源组件库，用于 ReactJS 和 Tailwind CSS，灵感来自 Material Design。

本博客使用的主题是 [Material Tailwind](https://www.material-tailwind.com/) 的 [Material Tailwind v3](https://www.material-tailwind.com/docs/v3/react/installation) 版本。


## 安装

使用 Astro 创建项目：

```bash
npm create astro@latest
```

安装 React 和 Tailwind CSS：

```bash
npx astro add react
```

安装 Tailwind CSS (注：使用tailwindcss v3)：

```bash
npx astro add tailwind
```

安装 Tailwind CSS 的 Typography 插件：

```bash
npx astro add @tailwindcss/typography
```

安装 Shiki 插件：

```bash
npx astro add shiki
```

安装 Material Tailwind：
```bash
npm install @material-tailwind/react
```
